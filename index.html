Rain On The Road

This task is called "Rain on the Road" and consists of a 5x8 matrix. The user needs to mark desired cells on the matrix to indicate the locations of a road. These marked cells will be colored black.

Once the user marks the road, the algorithm will analyze the matrix and color the potholes in blue. Potholes are defined as cells that are surrounded by black cells on both sides, which represent the road. The algorithm will color only those cells that meet this condition.

Finally, the algorithm will calculate how many cells remain uncolored, i.e., how many cells were neither marked as road nor colored as potholes.

In summary, the user marks the road, the algorithm identifies potholes based on neighboring cells, colors them, and then calculates the number of remaining uncolored cells.
